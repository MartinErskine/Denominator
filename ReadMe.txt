This is a Console Application.
Please extract and build the application to generate executables.

The Command Line exectuable will be in the generated BIN folder in the Denominator folder.

From a commandline in the root execurtable folder, enter:

Denominator(.exe) --c xx.xx --t xx.xx

Or

Denominator(.exe) --cost xx.xx --tendered xx.xx

The argument prefixes can be in any order (i.e. --t 20 --c 5.50) and can be either the single prefix or the full name.

...where xx.xx is the cost and the amount tendered.
